# **App Name**: DomiMatch

## Core Features:

- Aide Ménagère Listing: Display a scrollable list of available aides ménagères, each represented by a card.
- Aide Ménagère Card: Display the aide's photo, name, neighborhood, service type, and availability.
- Contact Button: Implement a button to contact the aide directly via WhatsApp.
- Data Fetching: Load aide data from static mock data to populate the application.
- Responsive Grid: Display aide cards in a responsive grid layout, adapting to different screen sizes.

## Style Guidelines:

- Primary color: Soft blue (#A0D2EB) to evoke trust and cleanliness.
- Background color: Very light grey (#F5F5F5) for a clean and modern look.
- Accent color: Light teal (#B0E57C) for interactive elements like the contact button.
- Body and headline font: 'PT Sans' for a modern and readable interface.
- Mobile-first responsive design ensuring optimal viewing experience on all devices.
- Simple, professional icons to represent service types and availability.
